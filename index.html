📁 Folder နှင့် File Management

Folder တစ်ခုဆောက်တော့မယ် ဆိုရင် mkdir နဲ့ ဆောက်လို့ရတယ် ။
mkdir folder_name
File တစ်ခု ဆောက်ချင်တယ် ဆိုရင်တော့ touch command နဲ့ ဆောက်လို့ရတယ် ။
touch filename.txt
Folder ဖျက် ချင်တယ်ဆိုရင် ကိုဖျက်ချင်တဲ့ Folder နာမည်ကို သိရမယ် ပြီးရင် rmdir command နဲ့ ဖျက်လို့ရတယ် ။ Folder Name ရဲ့ အနောက်မှာ ( / ) လေး ထည့်ဖို့ မမေ့ နဲ့ ၊ Tab Key နဲ့ တွဲ သုံးတယ် ဆိုရင်တော့ Auto ပါလာလိမ့်မယ် ။
rmdir folder_name/
Folder ပဲ ဖြစ်ဖြစ် File ပဲ ဖြစ်ဖြစ် ဘာဖြစ်ဖြစ် ဖျက်ချင်တယ် ဆိုရင်တော့ ( rm -rf ) command ကိုသုံးပြီး ဖျက်ပေးလို့ရတယ် ။ ဘာ error မှာမတက်ဘဲ နဲ့ကောင်းကောင်းမွန်မွန် ပျက်ချင်ရင် ဒီ Command ကိုသုံးရတယ် ။
rm -rf file_or_folder
rm -rf file1 file2 folder1 folder2
Terminal က ထွက်ချင်ရင် ( exit ) command ကိုသုံးရတယ်
exit

📋 File Operations - Copy & Move

File Copy - File တွေကို Copy ကူးချင်တယ်ဆိုရင် ( cp ) Command နဲ့ သုံးလို့ရတယ် ။ အရင် ဆုံး ကို ကူးချင်တဲ့ File ရှိတဲ့ နေရာကို သွား ရောက်ပြီ ဆိုရင် ကို ကူး ပြီးထည့်လိုက်ချင်တဲ့ Folder ကို ရွေးပြီးပြီး ထည့်ပေးရတယ် ။
cp source_file destination_path
လမ်းကြောင်းကို မှန်အောင်ရွေ့ ဖို့ လိုတယ် ။ ရွေ့ ချင်ရင် အရင်ဆုံး ( pwd ) command သုံးပြီး ကိုရွှေ့ချင်တဲ့ Folder ကဘယ်အောက်ရောက်နေလဲ ဆိုတာကို အရင် စစ် ။
pwd
File တွေကို ရွှေ့ ချင်ရင် ( mv ) Command ကိုသုံးရတယ် ။ ဒီဟလဲ အပေါ်က copy file လိုမျိုးပဲ တူတူပဲ ဒါကတော့ ဖိုင်တွေကို ကော်ပီ ကူးထည့်တာ မဟုတ်ပဲ အပြီး အပိုင် ရွေ့ချင်တဲ့ အခါမျိုးမှာ သုံးတယ် ။ ဒီ Command က Rename လုပ်ချင်ရင်လဲ သုံးလို့ရတယ် ။
mv source_file destination_path
နාမည် ချိန်းမယ် ဆိုရင် ( mv ကိုရဲ့မူရင်းရှိတဲ့ File name ကိုချိန်းချင်တဲ့ File name ) enter ခေါက်လိုက်။ ဒီလိုနည်းလမ်းနဲ့ နာမည်ချိန်းလို့ရတယ် ။
mv old_filename new_filename

👤 User Information

User Account ကြည့်ချင်တယ် ဆိုရင် ( whoami ) နဲ့ကြည့်လို့ရတယ် ။
whoami

📜 Command History

ရိုက်ခဲ့တဲ့ Command တွေကိုကြည့်ချင်တယ် ဆိုရင် ( History ) နဲ့ကြည့်လို့ရတယ် ။
history
အဲ့လို ရိုက်ခဲ့တဲ့ Command တွေကို ဖျတ်ချရင် ( HISTSIZE=0 ) command နဲ့ပြန်ဖျက်တယ် ။
HISTSIZE=0

🖥️ System Information

Kernel Version ကို ကြည့်မယ် ဆိုရင် ( uname -a )
uname -a
Ubuntu Version ကို ကြည့်ချင်တယ် ဆိုရင် ( cat /etc/os-release ) နဲ့ ကြည့်
cat /etc/os-release

🌐 Network & Hostname

Hostname ကို ချိန်းလို့ရတယ် အသေမဟုတ်ဘူး - Hostname ကို ချိန်းချင်ရင် ( sudo hostname ချိန်းချင်တဲ့ နာမည် ရေး Enter )
sudo hostname new_hostname
ifconfig ဆိုတာ IP Address တွေကိုကြည့်တာ ။ တကယ်လို့ ကို့ Ubuntu မှာ net-tools မသွင်းရသေးဘူး ဆိုရင်တော့ ဒီ ifconfig သုံးလိုက်ရင်တော့ error ပြပါလিမ့်မယ် ။
ifconfig

⏱️ System Uptime

uptime - ဆိုတာ ကို့ရဲ့ System ကြီး run နေတာ ဘယ်လောက်ရှိသွားပြီလဲဆိုတာ ကိုသွားကြည့်တာ
uptime

🔍 File Location

which - located File ကို့စတ်ထဲမှာ ဘယ် File တွေ ဘယ်နေရာမှာ ရှိလဲ ဆိုတာကို ကြည့်တာ
which command_name
type - ဒီ command ကလဲ which လို ပဲ သုံးလို့ရတယ် ။
type command_name

📚 Command Information & Help

whatis - ကို ရိုက်လိုက်တဲ့ command က ဘာလဲ သိချင်ရင် ဒီ Command ကိုသုံးပြီးကြည့် လို့ရတယ် ။
whatis command_name
info - ဒီ command က သပ်သပ် ထပ်ပြီးတော့ install လုပ်ရတယ် မရှိရင် ၊ သူက တစ်ချို့ command တွေ ဘာလုပ်လဲ ဘာအဓိပ္ပာယ်လဲ ဆိုတာကို သိချင်ရင် သူ့ကို သုံးရတယ် ဥပမာ ( info sudo ) ( info mv ) ဒီလိုမျိုးတွေ သုံးရတယ် သူ့ကို
sudo apt install info
info sudo
info mv

📊 File Analysis & Text Processing

file - ဘာဖိုင်လဲ ဆိုတာ ပြချင်ရင် ဒီ Command ကိုတွဲ သုံးရတယ် ။ ( file vv.txt ) ဒီလိုမျိုး ရေးပေးရတယ် ။
file filename.txt
wc - .txt file တွေကို ကြည့်တဲ့အခါမျိုးတွေမှာ သုံးတယ် ။ ဒီ option တွေနဲ့ တွဲသုံးတယ် Text File ထဲမှာ စာဘယ်နကြောင်း စားဘယ်နလုံးရှိလဲ ဆိုတာကို ကြည့်တဲ့ နေရာမှာ သုံးတယ် ။
wc filename.txt
word အရေအတွက် ကို ကြည့်ချင်ရင် ( -w ) option လေးကို ရေးပေးရတယ်
wc -w filename.txt
wc -l filename.txt
wc -c filename.txt
